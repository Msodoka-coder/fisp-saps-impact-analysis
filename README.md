# FISP and SAPS Impact Analysis

This project evaluates the impact of the Farm Input Subsidy Program (FISP) and Sustainable Agricultural Practices (SAPS) on:

- Agricultural productivity
- Food Consumption Score (FCS)
- Household Dietary Diversity Score (HDDS)

## Methods
- Descriptive statistics
- Multinomial probit model
- Bivariate probit robustness check
- Multinomial endogenous switching regression (MESR)

## Software
